# Week 2 Resources: Prompt Engineering & LLM Basics

This folder contains curated resources to deepen your understanding of prompt engineering, LLM behavior, tokenization, and practical templates you can reuse during labs and POCs.

Contents:
- `references.md` — Authoritative guides, blogs, papers, and docs
- `prompt-cheatsheet.md` — Dos and don'ts; quick patterns you can paste
- `example-prompts.md` — Reusable templates for common enterprise tasks
- `tokenizer-tools.md` — Token counting, pricing, and tokenizer tips
- `reading-list.md` — Longer-form content for deeper study

Tip: Open alongside the Week 2 lessons and labs for fast copy/paste and reference.
